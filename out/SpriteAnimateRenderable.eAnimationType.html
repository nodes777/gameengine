Namespace: eAnimationType

SpriteAnimateRenderable.eAnimationType

Assumption: first sprite in an animation is always the left-most element. Animations are always organized along the same row.
Properties:
Name Type Description
eAnimateRight number Animate from left to right, then restart to left
eAnimateLeft number Animate from right to left, then restart to right
eAnimateSwing number Animate left to right then animate backwards
